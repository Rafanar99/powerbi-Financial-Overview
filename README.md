# 📊 Financial Overview

A Power BI dashboard that provides a comprehensive view of the company's financial health — from revenue and profit to client concentration and payment methods — enabling stakeholders to monitor KPIs and make data-driven decisions.

## 🎯 Key Features

- 💰 **Revenue & Profit Analysis** — Total revenue, costs, expenses, and net profit visualized over time
- 🏆 **Top 5 Clients** — Highlights the clients with the highest revenue contribution
- 💳 **Payments by Type** — Breakdown of payment methods (credit card, bank transfer, etc.)
- 📈 **Monthly Profit Trends** — Identifies patterns and anomalies across months

## 🚀 How to Use

1. Click the dashboard link above to access the online version
2. Use the **date slicer** to filter by period
3. Explore the **revenue and profit cards** for a high-level financial summary
4. Check the **Top 5 Clients** visual to identify key accounts
5. Analyze the **Payments by Type** chart to understand payment distribution
6. Review the **Monthly Profit Trend** line chart to spot patterns over time
